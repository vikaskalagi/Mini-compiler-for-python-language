Mini Compiler FOR PYTHON LANGUAGE
----------------------------------

Our mini-compiler is built for a subset of the Python language (i.e only For and
IF ELSE statements). We have used tools such as yacc/bison, lex/flex and Python
scripts to build the complete compiler.


cmd to execute

make
./icg
python3 codegen.py<output_file.txt

//here output_file.txt contain the optimised ICG output
